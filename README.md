# Columbia Software Solutions (CSS)

Welcome to the team. This doc covers everything you need to get set up and start contributing.

---

## What we do

CSS is a student-run software club at Columbia where small teams build real software for real clients. You will work with an actual company or nonprofit, ship a real product in 8–10 weeks, and get mentored by an industry engineer through the whole thing.

---

## Tools

Set these up before your first sync.

**Slack** — this is our main communication tool. All project updates, questions, and coordination happen here. No group chats.
- Join the workspace: [https://columbiasoftware.slack.com]

**GitHub** — all code lives here under the CSS org.
- Org: [https://github.com/columbiasoftwaresolutions]

**Linear** — we track tasks and sprints here.
- Workspace: [https://linear.app/columbiasoftwaresolutions]
